Projects

May 2024 - Aug 2024

I was hired as a Research Assistant to develop a Gaussian Process (GP) ML model to estimate the age of S/Sq/Q-type asteroids. The GP model is trained with the reflectance data of said asteroids.

GP model was selected for this task as it has great potential even with a sparse training dataset like ours. I conducted a comprehensive study in order to select the most optimal model structure, which consists of a Convolutional Neural Network (CNN) based Feature Extractor, and Hadamard Multitask Regression which are built on top of the base GP model. The base GP elements are Constant Mean and Matern Kernel.

The Hadamard Multitask functionality is useful in our case because it can be used to train a separate initial model for each of the asteroid types, and to combine those models into one algorithm, which is optimized through training to benefit from the similarities between those models or asteroid types.

I'm proud of the results my model was able to achieve. It was comparable to a neural network model trained for the same task. More information can be seen in the future when our paper is published. In the mean time I have provided the link to my Github repository.

Main Skills Needed:

  • Mathematical Understanding of the Underlying GP and CNN Models
  • PyTorch/GPyTorch
  • Data Cleaning Pipeline
  • ML Optimization/Testing
